A clearer explanation

How surplus-based planning works in this prototype

Network effect concept

A system like this becomes more useful when more people participate and more infrastructure enters the network. If only a few people contribute very small amounts, the system has limited room to move. If participation grows, the network may have more utility, more scale, and more planning flexibility.

This is still not a promise. It simply explains the idea: more participation and more infrastructure can create more possible paths than a small isolated plan.
